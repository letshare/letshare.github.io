Grunt 只需要配置好两个文件 package.json Gruntfile.js

package.json指定插件和元信息。

Gruntfile.js配置任务和加载插件

task tip
grunt-responsive-images
生成多种尺寸的图片

task tip
grunt-contrib-imagemin
优化图片可以让你的页面更”轻”

speed tip
grunt-concurrent
同时跑一些比较慢的任务Sass、Coffee，可以显著的提供你的构建速度。

speed tip
grunt-newer
只有原文件在上一次任务之后改动过才重新运行任务。
